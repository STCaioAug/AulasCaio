2. Página Personalizada do Aluno

Ao fazer login, o aluno será redirecionado para uma página exclusiva, com as seguintes informações visíveis somente para ele:

Quantidade de aulas já realizadas

Aulas já agendadas

Horários disponíveis para agendamento de novas aulas

O aluno poderá:

Agendar novas aulas

Editar os horários das aulas agendadas

Visualizar os horários da semana, classificados como:

Verde: horários livres

Vermelho: horários reservados

Editar seus dados pessoais, incluindo:

Foto de perfil

Nome

E-mail

Senha de acesso

Importante: o aluno não pode ver o nome ou agendamento de outros alunos, apenas a disponibilidade geral.

3. Notificações para o Professor

Toda vez que um aluno agendar uma aula pelo sistema, uma notificação será enviada automaticamente para o professor (Caio).

Acesso exclusivo do administrador à página completa com todas as funcionalidades.
